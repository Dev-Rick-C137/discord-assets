## 📖 About

Welcome to **discord-assets** — a curated, organized collection of **official Discord Shop assets**: avatar decorations, nameplates, and profile decorations.

> ℹ️ **Note:** All assets in this repository are **official Discord Shop assets**. They are not custom-made or fan-created — this repo simply collects, organizes, and provides easy access to them.

## 🚀 Usage

All asset files are hosted on **Google Drive** — use the **Download** buttons in the table above to grab the category you need.

### Cloning this Repository

```bash
git clone https://github.com/Dev-Rick-C137/discord-assets.git
```

This downloads the repository itself; the actual decoration/nameplate files are on Google Drive (see table above).

Once downloaded, apply the asset directly in your Discord **User Settings**.

## 🤝 Contributing

Contributions are welcome! To add or improve assets:

1. Fork this repository
2. Create a new branch — `git checkout -b feature/new-assets`
3. Commit your changes — `git commit -m "Add new avatar decorations"`
4. Push to your branch — `git push origin feature/new-assets`
5. Open a Pull Request

Please make sure any assets you contribute are official Discord Shop assets and properly credited.
